## 작성 규칙
* root에 폴더를 생성하여 분류하고 파일명은 영문으로 만든다.
* 작은 것이라도 기록하고 매일 작성하는 실행력을 목표로 한다.
* 문서 생성은 마크다운 문법을 기반으로 한다. (.md파일)
* 관련된 출처는 정보가 포함된 외부 링크를 사용한다.
